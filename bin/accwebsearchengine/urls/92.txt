About twitter.com supported browsers Skip to main content Help Center About twitter.com supported browsers We officially support versions of the following browsers from the past year on twitter.com: Edge (https://microsoft.com/edge) Safari (https://www.apple.com/safari) Chrome (https://www.google.com/chrome) Firefox (https://www.mozilla.org/firefox) Any browser based on Webkit / Chromium engines (Opera, Samsung Internet, UCBrowser, etc.) If you use Twitter on an older version of the above browsers, browsers not listed, or a browser extension which modifies your user agent, you may be unable to use Twitter. In order to ensure the best possible Twitter experience, including all the latest features and security updates, please make sure your browser and operating system are up-to-date.
